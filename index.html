Data Diri Saya

Saya adalah seorang mahasiswa semester 4 yang masih pemula dalam pengembangan web di bidang frontend dan backend. Dalam pengembangan frontend, saya lumayan paham sedikit menggunakan HTML, CSS, dan JavaScript untuk membuat antarmuka pengguna yang responsif dan menarik.
Di sisi backend, saya juga lumayan paham dengan Python dan PHP, serta framework seperti Django, yang memungkinkan saya membangun aplikasi server-side yang efisien.
Kombinasi dari kedua keterampilan ini memungkinkan saya untuk mengembangkan aplikasi web penuh yang fungsional dan user-friendly, mulai dari desain antarmuka hingga pengelolaan basis data.
